О проекте

НЕРОРЕЧЬ

Программа которая помогает голосовому помощнику услышать вас.

Я написала её на языке Python используя библиотеку Speech recognition.

На картинке справа можно увидеть код и библеотека Python.

Как это работает?

  1. Пользователь делает запрос

    Для этого мы создали
    удобный интерфейс
  2. Программа используется
    специальные библиотеки

    Speech recognition-библеотека,
    позволяет создавать сценарии для
    доступа к микрофонам и обработки
    аудио файлов с нуля
  3. Обработка звука

    Программа распознаёт речь
  4. Преобразует запрос
    в голос и озвучивает

    Для преобразования текста
    в голос программа использует
    библиотеку SpeakerPy
